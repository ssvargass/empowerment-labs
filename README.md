## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation
- Docker
- Docker compose

```bash
$ yarn install
```

## Running the app with Docker (Suggested)
```bash
$ docker-compose up
```

## Dynamo Admin
[http://127.0.0.1:8001](http://127.0.0.1:8001)

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
